# Guess Number game 
This is a simple python game where the user tries to guess a random number .
## Features
 - Random number between 1-50
 - Users has 3 Chances to guess
 - Hints (too high/ too low )
# How to Run
 1. Install python
 2. Run the python file :
    python code.py
